# TajirConnect - Static Site

This is a static HTML, CSS, and JavaScript version of the TajirConnect website.

## Files

*   `index.html`: The main HTML file.
*   `style.css`: Contains all the styles for the website.
*   `script.js`: Handles fetching and displaying ad data from Firebase Firestore.

## Deployment

These files can be deployed to any static hosting service, such as GitHub Pages.
